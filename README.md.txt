 ...> Étude : Les relations entre le tabagisme et le cancer du poumon

 Objectif
Analyser les relations entre le tabagisme, les facteurs de mode de vie (âge, BMI, activité physique, etc.)
et le risque de cancer du poumon à partir d’un dataset de 4000 individus.

Technologies utilisées
- Python 3
- Pandas
- Matplotlib
- Seaborn
- NumPy
- SciPy
- Jupyter Notebook

Contenu du projet
- **rapport_de_pfm.pdf** : rapport de fin de module complet
- **analyse_tabagisme.ipynb** : notebook contenant les étapes d’analyse et de visualisation
- **data/smoking_cancer.csv** : dataset utilisé

Principales conclusions
- Le tabagisme actif est le principal facteur de risque.
- La consommation de cigarettes varie significativement selon le sexe.
- Le score de risque composite (RISK_SCORE) met en évidence la forte contribution du tabagisme.
